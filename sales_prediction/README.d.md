# CodeAlpha Sales Prediction

## Overview
This project is part of the CodeAlpha Data Science Internship.
It predicts product sales based on advertising spend using machine learning.

## Task
Task 2: Sales Prediction using Python

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Description
The model analyzes how advertising budgets across TV, Radio,
and Newspaper platforms affect sales outcomes.

## How to Run
1. Keep all files in one folder
2. Run: python sales_prediction.py
3. View evaluation metrics and graph

## Business Insight
Higher advertising spend generally leads to increased sales,
helping businesses optimize marketing strategies.
